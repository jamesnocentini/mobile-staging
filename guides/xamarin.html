Getting Started

The framework provides the interfaces for writing apps with Couchbase Lite. Starting from an empty Android project:

Setup Android Environment

  1. Create a new project from Xamarin Studio by clicking on New Solution:

  2. Select Android under the C# tab and choose Android Application as the project. Define an application project name and click OK

Add Couchbase Lite NuGet Package

  1. Download Couchbase Lite .NET package from NuGet by right clicking on the project within the Solution explorer section and selecting Add > Add NuGet Packages…

  2. From the Add Packages prompt, type Couchbase Lite in the top right search field. Select Couchbase Lite from the list of results and click Add Package. Xamarin Studio will add the packages and all the dependencies.

Verify Couchbase Lite Setup

  1. Click on the Package Console tab on the bottom right from within Xamarin Studio to see detailed information on the installed Couchbase.Lite package

  2. Verify the installation by seeing if you have the message Successfully added ‘Couchbase.Lite..’ to your project from within the Package Console
